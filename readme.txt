== Description ==

Smartart Accordion for Elementor is a custom Elementor widget that allows you to easily create an accordion-style content block with toggle functionality in your Elementor-powered WordPress site.

### Features:

- Customizable Accordion Content: Easily add and edit content using the WYSIWYG (What You See Is What You Get) editor for rich text formatting.
- Collapsible Accordion: Supports dynamic height adjustment based on the number of visible lines, offering a clean and compact accordion with expandable content.
- Toggle Buttons with Icons: Optional icons (open and close) can be added alongside the toggle button for enhanced visual appeal. Icons are customizable with size and color options.
- Dynamic Height Calculation: The accordion's collapsed state is dynamically adjusted based on the visible lines defined in the settings.
- Content Styling: Style options for the accordion content include background color, text color, padding, border radius, and box-shadow for custom looks.
- Button Styling: Customize the button's appearance, including text color, background color, hover effects, border radius, and padding. Button alignment options for left, center, and right positioning.
- Typography Control: Customize font styles, including size, weight, and family, for both accordion content and button text.
- Text Alignment: Flexibility to align text content within the accordion (left, center, or right).
- Mobile Responsive: Fully responsive accordion that adapts to different screen sizes for both desktop and mobile views.

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload `smart-elementor-accordion` to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Add the widget to any Elementor section by searching for "Smart Accordion" in the widget panel.

== Frequently Asked Questions ==

= Is this plugin compatible with the latest version of Elementor? =
Yes, the Smart Elementor Accordion widget is built specifically for Elementor and is compatible with recent versions of Elementor.

= Can I use the widget without Elementor Pro? =
Yes, this widget only requires the free version of Elementor.

= How do I translate the widget to my language? =
Simply place your `.po` and `.mo` language files in the plugin's `languages` folder. The plugin will load them automatically. Make sure your WordPress site language is set to the correct language in **Settings > General**.
